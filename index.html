Genshin Impact Tools

Simple tools for Genshin Impact without the extra $#@%. No Trackers, No Ads, No Gimmicks. It just works.

Tools

  • Achievements - (Outdated) Keep track of which achievements you've obtained and what's still to do.
  • Calculator - Use this calculator to see how much of each resource you will need to level your character from your current level to your end goal.
  • Schedule - A quick reference for resources and on which days they are available.
  • Tier List - Create and customize your own character ranking tier list.
  • Weapon Calculator - Use this calculator to see how much of each resource you will need to level your weapon.
  • Repo - All this is open source. Checkout the GitHub repo to view code/data or open issues/suggestions.
